# Serendib Spice Haven — Custom Odoo Templates

Production-ready, highly tailored HTML and inline CSS architecture deployment scripts built for embedding within Odoo Enterprise platform custom script areas. This repository contains the modular interface layers for the elite luxury spice brand, **Serendib Spice Haven**.

---

## 🌟 Brand Context & Aesthetic Blueprint
Inspired by high-altitude volcanic single-estate agriculture, monsoonal dark soils, and contemporary elite luxury global trade spaces. The layouts are optimized for absolute mobile legibility, precise section element separation, and strict grid tracking alignment to prevent element crowding loops on small viewports.

## 🛠️ Deployment Configuration Rules
1. All files feature strict global structural layout overrides explicitly designed to **suppress native Odoo footers** (`display: none !important;`) while safely preserving Odoo native header navigation menu layouts.
2. Graphic containers utilize responsive asset container frame systems (`.graphic-pane-frame`) that map clean `img` references with fallback broken-link handlers (`onerror="this.style.display='none'"`), allowing hassle-free web image uploads.
3. Forms are configured natively to interface straight out-of-the-box with the Odoo CRM capture route backend using `action="/website/form/"` and `data-model_name="crm.lead"`.
